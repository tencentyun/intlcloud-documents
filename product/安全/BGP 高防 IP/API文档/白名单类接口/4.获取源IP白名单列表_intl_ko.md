## API 설명
Anti-DDoS Advanced 서비스팩의 오리진 IP 화이트리스트 가져오기에 사용되는 API입니다.
프로토콜: `HTTPS`
도메인 이름: `bgpip.api.qcloud.com`
API 이름: `GetSrcWhiteIP`

## 요청 매개변수

| 매개변수 | 필수 여부 | 유형 | 설명 |
|---------|---------|---------|---------|
| bgpId | 예 | 문자열 | Anti-DDoS Advanced 서비스팩의 리소스 ID |
