## API説明
Anti-DDoS AdvancedのソースIPホワイトリストを取得するためのAPI。
プロトコル：`HTTPS`
ドメイン名：`bgpip.api.qcloud.com`
API名：`GetSrcWhiteIP`

## リクエストパラメータ

| パラメータ | 必須 | タイプ | 説明 |
|---------|---------|---------|---------|
| bgpId | はい | String | Anti-DDoS AdvancedのリソースID |
