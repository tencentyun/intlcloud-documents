## 1. 接口描述

接口请求域名： organization.tencentcloudapi.com 。

创建企业组织

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](https://cloud.tencent.com/document/api/850/38722)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：CreateOrganization |
| Version | 是 | String | 公共参数，本接口取值：2018-12-25 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| OrgType | 是 | Integer | 组织类型（目前固定为1） |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| OrgId | Integer | 企业组织ID|
| Nickname | String | 创建者昵称|
| Mail | String | 创建者邮箱|
| OrgType | Integer | 组织类型|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 创建企业组织

#### 输入示例

```
https://organization.tencentcloudapi.com/?Action=CreateOrganization
&OrgType=1
&<公共请求参数>
```
