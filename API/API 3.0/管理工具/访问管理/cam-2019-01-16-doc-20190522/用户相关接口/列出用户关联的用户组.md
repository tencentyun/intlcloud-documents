## 1. 接口描述

接口请求域名： cam.tencentcloudapi.com 。

列出用户关联的用户组

默认接口请求频率限制：20次/秒。

## 2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见 [公共请求参数](/document/api/598/33158)。

| 参数名称 | 必选 | 类型 | 描述 |
|---------|---------|---------|---------|
| Action | 是 | String | 公共参数，本接口取值：ListGroupsForUser |
| Version | 是 | String | 公共参数，本接口取值：2019-01-16 |
| Region | 否 | String | 公共参数，本接口不需要传递此参数。 |
| Uid | 是 | Integer | 子用户 UID |
| Rp | 否 | Integer | 每页数量。 |
| Page | 否 | Integer | 页码。 |

## 3. 输出参数

| 参数名称 | 类型 | 描述 |
|---------|---------|---------|
| TotalNum | Integer | 子用户加入的用户组总数|
| GroupInfo | Array of [GroupInfo](/document/api/598/33167#GroupInfo) | 用户组信息|
| RequestId | String | 唯一请求 ID，每次请求都会返回。定位问题时需要提供该次请求的 RequestId。|

## 4. 示例

### 示例1 列出用户关联的用户组
