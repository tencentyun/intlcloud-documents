若系统自动分配的公网地址不能满足业务需求时，您可以在控制台申请互联网通道公网 IP。本文将介绍如何在控制台申请及管理公网 IP。

## 公网 IP 生命周期
![](https://main.qcloudimg.com/raw/ebb4ee3321c34679d0970b95c6247dfb.svg)
1. [申请公网 IP](#creat)：在控制台申请公网 IP 后，系统将自动分配 IP 地址，公网 IP 状态流转为 “使用中”。
2. 状态为“使用中”的公网 IP 可以进行网段拆分和停用操作。
   - [网段拆分](#wdcf)：适用于需要细化网段的场景。操作后，公网 IP 状态仍为“使用中”。
   - [停用公网 IP](#tygw)： 操作后，公网 IP 状态流转为“已停用”。
3. 状态为“已停用”的公网 IP 可以进行启用和退还操作。
   - [启用公网 IP](#qygw)：操作后，公网 IP 状态流转为“使用中”。
   - [退还公网 IP](#thgw)：操作后，公网 IP 状态流转为“已退还”。
4. [找回公网 IP](#refound)：状态为“已退还”的公网 IP，在8天内可以找回，找回后的公网 IP 经系统重新分配，状态为“使用中”。

## 前提条件
在控制台申请互联网通道公网 IP，具体操作请参见[申请公网互联网通道](https://intl.cloud.tencent.com/document/product/216/39107)。

## 申请公网 IP
申请的公网 IP 将自动绑定所在地域的互联网专用通道。若某地域有多个通道时，各通道将共享申请的公网 IP。
1. 在左侧导航栏选择【互联网通道】>【通联网通道公网 IP】。
2. 在“互联网通道公网 IP”页面上方选择地域，然后单击【申请】。
3. 在“申请 IP 地址”对话框中配置以下信息。

<table>
<tr>
<th width="10%">字段</th>
<th>含义</th>
</tr>
<tr>
<td>协议类型</td>
<td>可选择申请 IPv4 或 IPv6。若需申请 IPv6 地址，互联网专用通道需要开启 IPv6。</td>
</tr>
<tr>
<td>地址类型</td>
<td><ul><li>若协议类型选择 IPv4：地址类型可选择 BGP 和静态，选择静态后，还需选择单线类型为电信、联通或者移动。</li><li>若协议类型选择 IPv6：地址类型仅可选择 BGP 类型。</li></td>
</tr>
<tr>
<td>数量</td>
<td>每个腾讯云账户最多可申请 512 个 IPv4 地址，每次申请至少4个。</td>

<td>申请 IPv6 地址时，可以自定义IPv6 地址的掩码，掩码最长为64位。</td>
</tr>

4. 单击【确定】。

## 网段拆分
网段拆分适用于需要细化网段的场景。对于 IPv4 地址，掩码小于30时，才能进行拆分；对于 IPv6 地址，掩码小于64时，才能进行拆分。具体步骤如下：
1. 登录[ 专线接入控制台](https://console.cloud.tencent.com/dc/dc)，并在左侧导航栏选择【互联网通道】> 【互联网通道公网 IP】。
2. 在“互联网专用通道公网 IP”页面上方选择地域。
3. 在互联网通道公网 IP 列表中，单击目标公网 IP 右侧操作列单击【网段拆分】。

4. 在网段拆分页面选择拆分数量，并勾选【确定网段拆分后网段无法合并】，单击【拆分】。
    >!请确保发布网段已变更为拆分后 CIDR，否则可能会导致网络中断，请谨慎操作。
    >

## 停用公网 IP
执行停用操作后，被选中的网段中的 4 个 IP 地址将无法访问公网，请谨慎操作。
1. 登录[ 专线接入控制台](https://console.cloud.tencent.com/dc/dc)，并在左侧导航栏选择【互联网通道】> 【互联网通道公网 IP】。
2. 在“互联网专用通道公网 IP”页面上方选择地域。
1. 在互联网通道公网 IP 列表中，单击目标公网 IP 右侧操作列单击【停用】。

2. 在“停用 IP 地址”对话框，单击勾选【确定停用以上 IP 地址】，然后单击【停用】，即可停用该网段中的4个 IP 地址。
		

<span id ="qygw"></span>
## 启用公网 IP
已停用的公网 IP 才能进行启用操作。启用操作如下：
1. 登录[ 专线接入控制台](https://console.cloud.tencent.com/dc/dc)，并在左侧导航栏选择【互联网通道】> 【互联网通道公网 IP】。
2. 在“互联网专用通道公网 IP”页面上方选择地域。
3. 在互联网通道公网 IP 列表中，单击目标公网 IP 右侧操作列单击【启用】。

4. 在“启用 IP 地址”对话框，单击勾选【确定启用以上 IP 地址】，然后单击【启用】，即可启用该网段中的4个 IP 地址。

<span id ="thgw"></span>
## 退还公网 IP
已停用的公网 IP 才能进行退还操作。
1. 登录[ 专线接入控制台](https://console.cloud.tencent.com/dc/dc)，并在左侧导航栏选择【互联网通道】> 【互联网通道公网 IP】。
2. 在“互联网专用通道公网 IP”页面上方选择地域。
1. 在互联网通道公网 IP 列表中，单击目标公网 IP 右侧操作列单击【退还】。

2. 在“退还 IP 地址”对话框，单击勾选【确定退还以上 IP 地址】，然后单击【退还】，即可退还该网段中的4个 IP 地址。

<span id ="refound"></span>
## 找回公网 IP
退还公网 IP 后8个自然日内可以找回，找回后的公网 IP 默认启用。
1. 登录[ 专线接入控制台](https://console.cloud.tencent.com/dc/dc)，并在左侧导航栏选择【互联网通道】> 【互联网通道公网 IP】。
2. 在“互联网专用通道公网 IP”页面上方选择地域。
1. 在互联网通道公网 IP 列表中，单击目标公网 IP 右侧操作列单击【找回】。

2. 在“找回 IP 地址”对话框，单击勾选【确定找回以上 IP 地址】，然后单击【找回】。
