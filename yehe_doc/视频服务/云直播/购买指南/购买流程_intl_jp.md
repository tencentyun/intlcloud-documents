### Cloud Streaming Services（CSS）の購入プロセス

Cloud Streaming Services（CSS）は、後払いをサポートするため、Tencent Cloudアカウントを登録し、CSSサービスをアクティブにするだけで、CSSを利用できます。

#### Tencent Cloudアカウントの登録
Tencent Cloudアカウントがすでに登録されている場合は、この手順をスキップしてください。Tencent Cloudアカウントが登録されていない場合は、ドキュメント[Signing up for a Tencent Cloud Account](https://intl.cloud.tencent.com/document/product/378/17985)を参照して登録できます。Tencent Cloudは、電子メールアドレス登録などの複数の方法をサポートしています。
#### 実名認証
Tencent Cloudで実名認証がすでに完了された場合は、この手順をスキップしてください。実名認証が完了されていない場合は、[Identity Verification Guide](https://intl.cloud.tencent.com/document/product/378/3629) を参照して認証を完了してください。個人の実名認証または法人の実名の認証を完了した後、CSSサービスのアクティベーションを申し込むことができます。
#### サービス申し込み
[CSS Console](https://console.cloud.tencent.com/live)ページに入って、【Apply for Activation】をクリックします。正常に送信されると、CSSの機能を直接使用できます。Cloud Streaming Servicesサービスは、デフォルトで日次トラフィック課金となります。
