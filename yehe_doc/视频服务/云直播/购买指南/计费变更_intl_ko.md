>
>- Tencent Cloud의 주요 고객에 해당하는 경우, 비즈니스 매니저에게 문의하여 과금 방식을 변경할 수 있습니다.
>- 서비스 이용 중, 과금 방식에 대해 의문사항이 있는 경우 과금 방식을 다시 선택할 수 있습니다. 일별 트래픽 과금과 대역폭 과금 방식 간 전환 시, 전환 후 익일부터 적용됩니다. 일별 과금 방식을 월별 과금 방식으로 변경하는 경우 심사 통과 후 익일부터 적용되며, 월별 과금 방식을 일별 과금 방식으로 변경하는 경우 심사 통과 후 익월부터 적용됩니다.

## 변경 방법
1. [CSS 콘솔](https://console.cloud.tencent.com/live)에 로그인한 후, 왼쪽 메뉴에서 [Overview]를 클릭합니다. 개요 페이지 상단에 아래 그림과 같이 현재 과금 방식이 표시됩니다. 일별 트래픽 과금과 일별 대역폭 과금 방식 간 전환이 필요한 경우 [전환]을 클릭하여 변경할 수 있습니다.
![](https://main.qcloudimg.com/raw/e6552f6fd812d6d2d16ebc5a0035432a.png)

2. 일별 트래픽 과금 방식을 일별 대역폭 과금 방식으로 전환할 경우 [Confirm]을 클릭합니다.
![](https://main.qcloudimg.com/raw/ddbf3b586f7a536b72db69a5f432d3b9.png)

>
>- 과금 방식을 일별 대역폭 과금 방식으로 전환한 후에는 라이브 방송 트래픽 패키지가 차감되지 않으며, 트래픽 패키지 차감이 필요한 경우 일별 트래픽 과금 방식으로 다시 전환해야 합니다.
>- 현재 과금 방식이 월별 과금이거나 월별 과금으로 변경이 필요한 경우 비즈니스 매니저에게 문의해 주십시오.
