## 주의 사항
CSS에서는 기본적으로 테스트 도메인 `xxxx.livepush.myqcloud.com`을 제공하며, 해당 도메인을 통해 푸시 스트리밍을 테스트할 수 있습니다. 정식 비즈니스에 해당 도메인을 푸시 도메인을 사용하는 것은 권장하지 않으며, 해당 푸시 스트리밍은 삭제할 수 없습니다.

## 전제 조건
1. [Cloud Streaming Services](https://intl.cloud.tencent.com/product/css)가 활성화된 상태여야 합니다.
2. [외부 도메인 추가](https://intl.cloud.tencent.com/document/product/267/35970)가 되어 있어야 합니다.



## 도메인 조회
[도메인 이름 관리](https://console.cloud.tencent.com/live/domainmanage) 페이지에서 추가 완료한 도메인의 CNAME 설정 상태, 소속 유형, 시나리오, 리전, 현재 상태, 도메인 활성화 시간 및 만료 시간을 확인할 수 있습니다. 도메인 상세 정보를 조회하고 싶은 경우, 조회할 도메인 또는 해당 도메인의 오른쪽 [관리]를 클릭하면 도메인 정보를 확인할 수 있습니다.
![](https://main.qcloudimg.com/raw/b26af663a41a018e6bda8920c65e2dca.png)

## 도메인 설정
푸시 도메인 설정이 필요한 경우 [푸시 스트림 설정](https://intl.cloud.tencent.com/document/product/267/31059)을 참고하십시오.
재생 도메인 설정이 필요한 경우 [재생 설정](https://intl.cloud.tencent.com/document/product/267/31058)을 참고하십시오.


## 도메인 비활성화
도메인을 비활성화하고 싶은 경우, 비활성화할 도메인의 오른쪽에 있는 [비활성화]를 클릭하면 현재 도메인의 비활성화 여부를 확인하는 팝업창이 표시됩니다. [확인]을 클릭하면 해당 도메인 상태가 **"활성화"**에서 **"비활성화**로 변경되며, 이는 도메인 비활성화가 완료되었다는 의미입니다.
>? 도메인 비활성화 후 해당 도메인은 액세스할 수 없으며, 다시 활성화 후 정상적으로 액세스 할 수 있습니다. 재생 도메인과 푸시 도메인 작업도 동일합니다.

![](https://main.qcloudimg.com/raw/16c8c3f389c8e3a3c06c77e3c3ad8a23.png)

## 도메인 활성화
비활성화한 도메인을 다시 활성화하고 싶은 경우, 활성화할 도메인 오른쪽에 있는 [활성화]를 클릭합니다. 도메인 상태가 ‘비활성화’에서 ‘활성화’로 변경되며, 이는 도메인 활성화가 완료되었다는 의미입니다.
![](https://main.qcloudimg.com/raw/aff5677a3fedc365b09ce6991afe2b3a.png)


## 도메인 삭제
도메인을 삭제하고 싶은 경우, 해당 도메인 오른쪽에 있는 [삭제]를 클릭하면 현재 도메인의 삭제 여부를 확인하는 팝업창이 표시됩니다. [확인]을 클릭하면 도메인이 삭제됩니다.
![](https://main.qcloudimg.com/raw/16c8c3f389c8e3a3c06c77e3c3ad8a23.png)
