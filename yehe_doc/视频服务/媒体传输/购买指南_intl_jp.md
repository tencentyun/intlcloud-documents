###  課金説明

StreamLinkコンソールはストリームベースで管理を行い、ストリームごとに1つの転送リンクを利用します。StreamLinkコンソールにより、迅速的かつ安定的なビデオストリームメディアの転送を実現でき、さらに転送中のビデオストリームに対してオールラウンドな品質監視を行うことができます。StreamLinkの課金項目には以下の3つが含まれています。

| 課金項目 | 課金の説明 | 支払い方法 |
| -----------------| ---| --------- |
| 転送料金 |転送リンク利用時の課金です。発生した料金は各転送リンクのストリームの当日のピーク帯域幅の累計に応じて課金されます | 後払い-日次決済 |
| マシン運用料金 |マシンによる転送時の課金です。発生した料金は当日のタスク数と利用時間に応じて時間単位で課金されます | 後払い-日次決済 |
| 転送・プッシュトラフィック料金 |発生した料金は当日プッシュしたエリアとトラフィックに応じて課金されます | 後払い-日次決済 |

### 転送料金

#### 課金価格

StreamLinkはユーザーがコンソールで転送リンクを作成することを許可し、当日実際に使用した帯域幅ピーク値に応じて清算します。

| 課金項目   | 課金方式           | 価格          |
| -----------------| ---| --------- |
| 転送料金 |1日あたりのアウトピーク帯域幅に応じて課金されます。 | 7.6923米ドル/Mbps/日 |

#### 課金例

お客様は2022年1月1日にStreamLinkサービスを使用し、A、Bの2つのストリームを作成し、Aストリームが昼の12時にピーク帯域幅10Mbpsに達し、Bストリームが夜の22時にピーク帯域幅10Mbpsに達したとします。このとき、2022年1月2日に支払うライブストリーミングトラフィック請求書は以下のとおりとなります。
1日あたりの転送料金 = 7.6923（米ドル/Mbps/日）× 20（Mbps）= 153.846米ドル。



### マシン運用料金

#### 課金価格

StreamLinkサービス使用におけるマシン運用料金の課金方式は以下のとおりです。

| 課金項目     | 課金方式                               | 価格             |
| ---------- | -------------------------------------- | ---------------- |
| マシン運用料金 | マシン運用時間料金はストリームごと、時間ごとに料金がかかります。 | 0.25美元/ストリーム/時間 |


>!
> - ストリーム数はinputおよびoutput数に応じて計算されます。inputストリームが1個、outputストリームが3個の場合、4個のストリームに応じてマシン運用時間が計算されます。
> - マスター、スレーブストリームを含む1つのタスクは、2個のストリームに応じてマシン運用時間が計算されます。
> - 課金は時間単位で計算されます。1時間に満たない使用時間は1時間として計算され、精算サイクルは日次決済で計算されます。

#### 課金例

お客様は2022年1月1日にStreamLinkサービスを使用し、inputが1個、outputが2個あるタスクを作成しました。当日の使用時間は6時間です。このとき、2022年1月2日に支払うライブストリーミングトラフィック請求書は以下のとおりとなります。

1日単位のマシン運用料金=3（ストリーム数）× 6（時間） × 0.25（米ドル/時間）= 4.5米ドル。
