Tencent Effect SDK には全部で10のパッケージがあり、10のパッケージは、[Aシリーズベーシックパッケージ](#setA) と[Sシリーズハイグレードパッケージ](#setS)という2つのシリーズに分かれています。シリーズとパッケージの違いによって対応する機能が異なります。各パッケージのサポートする機能の詳細については次の表をご覧ください。その他のダウンロードの説明については、[SDKダウンロード](https://intl.cloud.tencent.com/document/product/1143/45377)をご参照ください。
