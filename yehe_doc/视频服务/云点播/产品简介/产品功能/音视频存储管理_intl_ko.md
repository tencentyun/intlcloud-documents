## 제품 기능
### 오디오/비디오 업로드
로컬 파일 시스템, URL 또는 API를 통해 파일을 업로드할 수 있습니다. VOD는 검사점 재시작, 대용량 파일 업로드, 다중 복제 백업을 지원합니다. 자세한 내용은 [미디어 업로드 개요](https://intl.cloud.tencent.com/document/product/266/9760)를 참고하십시오.
업로드에 지원되는 파일 형식은 다음과 같습니다.

* 비디오: WMV, RM, MOV, MPEG, MP4, 3GP, FLV, AVI, RMVB, TS, ASF, MPG, WEBM, MKV, M3U8, WM, ASX, RAM, MPE, VOB, DAT, MP4V, M4V, F4V, MXF, QT, OGG.
* 오디오: MP3, M4A, FLAC, OGG, WAV, RA, AAC, AMR.
* 썸네일: JPG, JPEG, PNG, GIF, BMP, TIFF, AI, CDR, EPS, TIF.

### 오디오/비디오 스토리지
VOD는 아키텍처와 장치 전반에 걸쳐 중복성을 갖춘 데이터 스토리지를 지원합니다. 기존 아키텍처를 능가하는 99.9%의 내구성을 제공합니다. 또한 요구 사항을 충족하기 위해 여러 스토리지 유형을 지원합니다.
* 스토리지 유형: 표준 스토리지, 표준IA 스토리지, 아카이브, 딥 아카이브.
* 스토리지 비용 절감: 맞춤형 규칙 기반 스토리지 정책, 만료된 미디어 파일 자동 삭제, 실시간 이미지 처리.

### 미디어 자산 관리
VOD 콘솔 또는 API를 통해 카테고리 및 라벨 추가 등의 작업을 수행하여 미디어 파일을 관리할 수 있습니다. 이러한 파일의 모든 정보를 내보낼 수 있습니다.

- 미디어 정보의 빠른 편집
- 여러 필터로 미디어 정보 검색
- 미디어 파일 처리, 트랜스코딩, 워터마크, 썸네일 설정, 스크린 캡쳐
- CSV/JSON 형식으로 미디어 정보 내보내기
- WeChat QR 코드를 통해 미디어 공유
- 비디오에 대한 여러 수준의 카테고리 및 레이블 설정
- 테이블별 미디어 관리, 온라인 미리보기, 다운로드
- 여러 미디어 스토리지 유형 지원
- 블랙리스트/화이트리스트 링크 도용 방지 설정
