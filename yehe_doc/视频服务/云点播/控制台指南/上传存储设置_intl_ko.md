## 작업 시나리오
VOD의 스토리지 업로드 설정에는 카테고리 관리 및 스토리지 리전 기능이 포함되어 있어 콘솔의 파일을 더욱 편리하게 관리할 수 있습니다.

## 카테고리 관리 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod/overview)에 로그인한 후, 왼쪽 메뉴에서 [스토리지 업로드 설정]>[카테고리 관리]를 선택하여 ‘카테고리 관리’ 페이지로 들어갑니다.
![](https://qcloudimg.tencent-cloud.cn/raw/95d60acb6d5c9bbfbb4c602ef27f5b4e.png)
2. [카테고리 추가]를 클릭한 후 ‘카테고리 추가’ 팝업창에 카테고리 이름을 입력하고 [확인]을 입력하여 카테고리를 추가합니다.
3. 새로 추가한 카테고리는 이 페이지의 카테고리 리스트에 표시되며, 여기에서 타깃 카테고리 기록의 이름을 바꾸거나 삭제 또는 하위 카테고리 추가를 할 수 있습니다.
	- 이름 변경: 타깃 카테고리의 이름을 클릭하면 이름 우측에 편집 아이콘이 나타나며, 아이콘을 클릭하면 카테고리 이름을 수정할 수 있습니다.
	- 삭제: 타깃 카테고리가 있는 행의 [삭제]를 클릭하면 카테고리가 삭제됩니다. 하위 카테고리가 포함되어 있는 경우에는 먼저 하위 카테고리를 삭제해야 합니다.
	- 하위 카테고리 추가: 타깃 카테고리가 있는 행의 [하위 카테고리 추가]를 클릭하면 ‘하위 카테고리 추가’ 팝업창이 뜨고, 여기에 카테고리 이름을 입력한 다음 [확인]을 클릭하면 하위 카테고리가 추가됩니다.

>?
>- 카테고리 이름은 중국어, 영어, 숫자, 소괄호를 지원하며 최대 64자까지 입력 가능합니다.
>- 카테고리 관리에서는 API를 통한 파일 조회, 수정, 삭제 및 연결을 지원합니다.
>- 카테고리 관리는 4단계의 카테고리 구조를 지원하며, 4단계 카테고리에는 하위 카테고리를 추가할 수 없습니다.
>- 카테고리 추가가 완료되면 콘솔의 미디어 자산 관리에서도 비디오 파일의 카테고리를 수정할 수 있습니다. 자세한 내용은 [비디오 카테고리 수정](https://intl.cloud.tencent.com/document/product/266/33893)을 참고하십시오.
>- 카테고리 관리에는 기본적으로 ‘기타’라는 카테고리가 있으며, 이름을 변경하거나 삭제하거나 하위 카테고리를 추가할 수 없습니다. 카테고리가 삭제되면 해당 파일이 ‘기타’로 분류됩니다.

##스토리지 리전 단계
1. [VOD 콘솔](https://console.cloud.tencent.com/vod/overview)에 로그인한 후, 왼쪽 메뉴에서 [스토리지 업로드 설정]>[스토리지 리전]을 선택하여 ‘스토리지 리전’ 페이지로 들어갑니다.
![](https://qcloudimg.tencent-cloud.cn/raw/f6239d2a24ad590e9a67fd7e61b2f6a2.png)
2. 타깃 리전이 있는 행의 상태 버튼을 클릭하여 해당 리전을 활성화합니다. 활성화된 리전은 기본 리전으로 설정할 수 있습니다.
>?
>- 여러 리전을 활성화할 수 있으나 그 중 하나만 기본 리전으로 설정할 수 있습니다.
>- 리전 활성화 시, 새로운 리전의 구성이 완료되기 전까지는 해당 리전에서 작업을 실행할 수 없습니다. 리전이 활성화된 후 적용되기까지 5 - 10분이 소요되며, **한번 활성화된 리전은 비활성화할 수 없습니다**.

#### 스토리지 규칙
- 다른 리전이 활성화되지 않은 경우, 모든 파일이 기본적으로 기본 리전으로 전송됩니다.
- 다른 리전이 활성화되어 있는 경우.
	- IP 소재지가 **활성화된 리전의 근처 업로드 스토리지 범위 내**에 있는 경우, IP에 따라 파일이 근처의 활성화된 리전으로 업로드됩니다.
	- IP 소재지가 **활성화된 리전의 근처 업로드 스토리지 범위 내**에 있지 않을 경우, VOD가 파일을 기본 리전에 업로드합니다.

스토리지 리전에 대한 더 자세한 내용은 [업로드 품질 최적화 사례](https://intl.cloud.tencent.com/document/product/266/37548)을 참고하십시오.
