腾讯移动推送服务提供包年包月和按量计费两种计费模式，本文描述两种不同计费模式的定价策略和折扣策略。
>! 此产品的包年包月售卖模式正在内测中，如有需要，请[联系商务](https://intl.cloud.tencent.com/contact-sales)。

## 名词解释
- 日活跃用户(DAU)：当日 SDK 累计联网的去重设备数（应用前台在线或后台进程在线时 SDK 会连接至 TPNS 服务器）；
- 日活跃用户(DAU)月峰值：当月内每天的日活跃用户(DAU)取最大值；
- 服务接入点：移动推送服务集群所在地。因不同地域对数据出入境有严格限制，可在控制台创建产品时选择您的应用服务接入点；

## 包年包月
### 注意事项
- 包年包月计费单位为「1万日活跃用户(DAU)」，您在购买时至少购买1万日活跃用户(DAU)对应的服务；

### 定价说明
由于不同服务接入点服务成本不同，故定价也存在相应差异。

| 服务接入点 | 单价（美元 / 1万日活跃用户(DAU) / 月） |
| ---------- | --------------------------------- |
| 广州       | 42.237                            |
| 中国香港   | 56.316                            |
| 新加坡     | 56.316                            |

### 折扣说明
| 日活跃用户(DAU)月峰值 | 购买时长：1 ～ 5 个月 | 购买时长：6 ～ 11 个月 | 购买时长：1 年 | 购买时长：2 年 |
| ------------------ | --------------------- | ---------------------- | -------------- | -------------- |
| 1万 ～ 49万        | 无                    | 无                     | 无             | 无             |
| 50万 ～ 199万      | 9 折                  | 8 折                   | 6.5 折         | 6 折           |
| 200万及以上        | 7 折                  | 6.5 折                 | 6 折           | 5 折           |

### 计算示例
**计费示例1**
- 若服务接入点为新加坡。
- Android 端 App 日活跃用户(DAU)月峰值为 60万 。
- 套餐时长为 12 个月。
则套餐总费用为：月费用 * 时长 * 折扣 = （ 60  * 56.316 ） * 12 * 0.65 = 26355.7944 美元；

**计费示例2**
- 服务接入点为广州
- iOS 端 App 日活跃用户(DAU)月峰值为10万
- 套餐时长为1个月
则套餐总费用为：月费用 x 时长 x 折扣 =（ 10 x 42.237 ）x 1 x 1 = 422.37 美元；

## 按量计费

### 定价说明

- 按量计费按天结算时：
	- 日活跃用户(DAU) <= 1000：当日不产生扣费。
	- 1000 < 日活跃用户(DAU) <= 1万：按照下方固定价结算并扣费。
	- 日活跃用户(DAU) > 1万：按照下方线性价结算并扣费。
	- 连续使用时长从该应用第一次开通按量计费日开始计算，使用中途停服或销毁不会清空连续使用时长（若结算时日活跃用户(DAU) <= 1000 未产生扣费，则该天不计入连续使用时长中）。
	

由于不同服务接入点服务成本不同，故定价也存在相应差异，具体定价请查阅下表：。

#### 1000 < 日活跃用户(DAU) <= 1万：固定价

| 服务接入点 | 单价（美元 / 天） |
| ---------- | ----------------- |
| 广州       | 1.6               |
| 中国香港   | 2.14              |
| 新加坡     | 2.14              |

>! 
>- 为让用户以低价享受优质服务，2021年01月04日起，按量计费的最低收费标准由5万设备量级降至1万：
>- 我们将为2021年01月04日之前接入按量计费且设备量级在5万以下的客户，在2020年01月04日当天自动按新资费计费，费用门槛更低更优惠。

#### 日活跃用户(DAU) > 1万 ：线性价
| 服务接入点 | 单价（美元 / 日活跃用户(DAU) / 天） |
| ---------- | ------------------------------ |
| 广州       | 0.000160                       |
| 中国香港   | 0.000214                       |
| 新加坡     | 0.000214                       |

### 折扣说明
| 连续使用时长（天） | 折扣   |
| :----------------- | :----- |
| 1 ~ 180            | 无     |
| 181 ~ 360          | 8 折   |
| 361 ~ 720          | 6.5 折 |
| > 720              | 6 折   |

### 计算示例
- 若服务接入点为新加坡且日活跃用户(DAU) <= 1万 ，则每日费用固定为 2.14 美元；
- 若服务接入点为新加坡且日活跃用户(DAU)为7万：
	- 若已连续使用90天，则无折扣，每日费用为 70000 * 0.000214 = 14.98 美元；
	- 若已连续使用200天，则对应 8折折扣，每日费用为 70000 * 0.000214 * 0.8 = 11.984 美元；
