
>## (1) IM SDK 오류 코드
>>?Web SDK 오류 코드에 대한 자세한 내용은 [오류 코드 대조표](https://web.sdk.qcloud.com/im/doc/en/module-ERROR_CODE.html)를 참고하십시오. 
>
>### 일반 오류 코드
>
>| 오류 코드 | 설명                                                         |
>| --------- | ------------------------------------------------------------ |
>| 4001      | 파일 메시지 실패: 해당 URL이 IM에서 할당한 스토리지 리소스가 아닙니다. |
>| 4002      | 파일 메시지 실패: 내부 서버 네트워크 오류입니다.             |
>| 4003      | 이미지 메시지 실패: CI에서 잘못된 형식을 반환했습니다.       |
>| 4004      | 이미지 메시지 실패: 잘못된 이미지 형식입니다.                |
>| 4005      | 파일 메시지 실패: 규정 위반 또는 차단된 파일은 보낼 수 없습니다. |
>| 6015      | 작업이 진행 중입니다. API 호출에 대한 제어를 최적화하십시오. 예를 들어 첫 번째 초기화 작업이 콜백되기 전에 다른 초기화 작업을 수행하면 시스템에서 이 오류 코드를 반환합니다. |
>| 6017      | 잘못된 매개변수입니다. 잘못된 매개변수를 찾으려면 오류 정보를 확인하십시오. |
>| 6022      | 로컬 IO 작업 오류입니다. 읽기/쓰기 권한이 있는지 또는 디스크가 꽉 찼는지 확인하십시오. |
>| 6027      | 잘못된 JSON 형식입니다. 특정 필드를 찾으려면 오류 정보를 확인하십시오. |
>| 6028      | 메모리가 부족합니다. 메모리 누수가 발생할 수 있습니다. iOS 플랫폼의 Instrument 툴 또는 Android 플랫폼의 Profiler 툴을 사용하여 메모리 사용량이 많은 위치를 분석하고 식별합니다. |
>| 6001      | PB 파싱에 실패했습니다. 내부 오류입니다.                     |
>| 6002      | PB 직렬화에 실패했습니다. 내부 오류입니다.                   |
>| 6013      | IM SDK가 초기화되지 않았습니다. IM SDK가 초기화되고 콜백을 통해 응답이 반환된 후 다시 시도하십시오. |
>| 6005      | 파일 손상으로 인해 로컬 데이터베이스를 로딩하지 못했습니다.  |
>| 6019      | 로컬 데이터베이스에 대한 작업이 실패했습니다. 이 오류는 일부 디렉터리에 대한 권한 부족 또는 데이터베이스의 파일 손상으로 인해 발생할 수 있습니다. |
>| 7001      | 크로스 스레드 오류입니다. 스레드 간 작업은 지원되지 않습니다. 내부 오류입니다. |
>| 7002      | TinyId가 비어 있습니다. 내부 오류입니다. UserID가 존재하고 유효한지 확인하십시오. |
>| 7003      | 잘못된 UserID입니다. UserID는 비워둘 수 없으며 길이가 최대 32바이트인 인쇄 가능한 ASCII 문자(0x20-0x7e)여야 합니다. |
>| 7004      | 파일이 존재하지 않습니다. 파일 경로가 올바른지 확인하십시오. |
>| 7005      | 파일 크기가 제한을 초과했습니다. 업로드된 파일의 최대 허용 크기는 100MB입니다. |
>| 7006      | 파일이 비어 있습니다. 파일은 0바이트일 수 없습니다. 이미지, 오디오, 비디오 또는 문서를 업로드할 때 파일이 올바르게 생성되었는지 확인하십시오. |
>| 7007      | 파일을 열지 못했습니다. 파일이 존재하는지 또는 독점적으로 열렸는지 확인하여 SDK에서 파일을 열 수 없습니다. |
>| 7008      | API 호출 빈도가 한도를 초과합니다. 자주 호출되는 API를 확인하고 피하십시오. |
>| 7009      | SDK가 실행되는 동안 종료됩니다. 예를 들어 로그인 시 unInit를 호출하여 SDK 실행을 종료합니다. |
>| 7010      | database 작업이 실패했습니다.                                |
>| 7011      | 쿼리한 데이터가 database에 존재하지 않습니다.                |
>| 7012      | SDK에서 내부 오류가 발생했습니다.                            |
>| 7013      | API가 지원되지 않습니다. 프리미엄 에디션으로 업그레이드하십시오. |
>| 7014      | 잘못된 요청입니다. API 호출이 규정을 준수하는지 확인하십시오. |
>| 7015      | SDK의 로컬 콘텐츠 조정이 민감한 단어에 히트되었습니다.       |
