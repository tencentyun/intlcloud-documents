### 사례: 표준IA 스토리지 용량 비용 + 표준IA 스토리지 데이터 검색 비용 + 표준IA 스토리지 요청 비용 + 공인 네트워크 다운스트림 트래픽 비용

사용자 B가 2020년 11월 1일 표준IA 스토리지 유형의 5GB 데이터를 광저우 리전의 COS 버킷에 업로드하고, 둘째 날 CDN을 활성화하지 않은 상태에서 공용 네트워크를 사용해 해당 데이터를 읽었고, 100번의 요청이 발생했으며, 다른 시간에는 아무 작업도 하지 않았다고 가정합니다. 이 경우 스토리지 용량 비용, 요청 비용은 월 결산되고 트래픽 비용은 일 결산됩니다. 이에 따라 2020년 11월 3일 공인 네트워크 다운스트림 트래픽 비용이 결산되며 2020년 12월 1일 표준IA 스토리지 용량 비용, 표준IA 스토리지 데이터 검색 비용, 표준IA 스토리지 요청 비용이 결산됩니다. 종량제 방식에 따른 비용 분석은 다음과 같습니다.

- 종량제:
 - 표준IA 스토리지 용량 비용 = 0.018USD/GB/월 x 5GB = 0.09USD
 - 표준IA 스토리지 데이터 검색 비용 = 0.002USD/GB x 5GB = 0.01USD
 - 표준IA 스토리지 요청 비용 = 0.01USD/1만 회 x 1만 회 = 0.01USD
 - 공인 네트워크 다운스트림 트래픽 비용 = 0.1USD/GB x 5GB = 0.5USD

위 금액을 모두 합산하면 전체 11월 사용자 B의 총 요금은 0.09 + 0.01 + 0.01 + 0.5 = 0.65USD입니다.
