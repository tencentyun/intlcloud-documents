## 2. 云数据库 Redis 的计费模式
云数据库 Redis 的计费模式为**按量计费**。您需要全面了解云数据库 Redis 的计费模式，有利于您选择最优的计费方案。详情请参见 [计费概述](https://intl.cloud.tencent.com/document/product/239/31954)。

-----

## 3. 新手入门
#### 3.1 创建 Redis 实例
在使用云数据库 Redis 之前，您需要注册腾讯云账号并且购买云数据库 Redis 服务。详情请参见 [创建 Redis 实例](https://intl.cloud.tencent.com/document/product/239/37712)。


#### 3.2 连接 Redis 实例
您可通过云服务器，以内外网两种不同的方式访问云数据库 Redis。详情请参见 [连接 Redis 实例](https://intl.cloud.tencent.com/document/product/239/9897)。
