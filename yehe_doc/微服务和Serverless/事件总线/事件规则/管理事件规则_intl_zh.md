本文为您介绍如何查看、编辑和删除事件规则。


## 前提条件

已 [创建事件规则](https://intl.cloud.tencent.com/document/product/1108/42289)。


## 操作步骤

### 查看事件规则

您可以通过以下操作查看事件集的详细信息。

1. 登录 [事件规则控制台](https://console.cloud.tencent.com/eb/rule)。
2. 在顶部菜单栏，选择地域与事件集。
3. 在事件规则列表单击需要查看的事件规则。
4. 事件规则页面显示事件规则的基础信息、事件匹配信息、事件目标等信息。

### 编辑事件规则

1. 登录 [事件规则控制台](https://console.cloud.tencent.com/eb/rule)。
2. 在顶部菜单栏，选择地域与事件集。
3. 在事件规则列表单击需要编辑的事件规则。
4. 事件规则页面可编辑基础信息、事件匹配、删除/新增事件目标等内容。

### 删除事件规则

1. 登录 [事件规则控制台](https://console.cloud.tencent.com/eb/rule)。
2. 在顶部菜单栏，选择地域与事件集。
3. 在事件规则列表找到您需要删除的事件规则，在其右侧操作栏下单击**删除**。
4. 在弹出的确认窗口中确认删除即可。
