Dokumen ini menjelaskan cara mengubah nama, tag atau informasi lain dari VPC.

## Petunjuk
1. Login ke [Konsol VPC](https://console.cloud.tencent.com/vpc).
2. Pilih wilayah VPC di bagian atas halaman **VPC** (VPC).
3. Klik ikon edit di sebelah nama VPC untuk mengubahnya.
4. Klik ID VPC untuk mengakses halaman **Basic Information** (Informasi Dasar).
5. Tag digunakan untuk mengidentifikasi dan mengelola sumber informasi. Anda dapat mengeklik ikon edit untuk menambah atau menghapus tag.
